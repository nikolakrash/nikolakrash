# TeleBoosting - Маркетинговый сайт

Маркетинговый сайт для продажи платформы TeleBoosting - мощного инструмента для автоматизации работы с Telegram аккаунтами.

## Структура проекта

```
telebooster-sait/
├── index.html          # Главная страница
├── static/
│   ├── css/
│   │   └── style.css   # Стили сайта
│   ├── js/
│   │   └── main.js     # JavaScript для интерактивности
│   └── videos/
│       └── demo.mp4     # Демонстрационное видео
└── README.md           # Этот файл
```

## Запуск сайта

### Вариант 1: Простой HTTP сервер (Python)

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Затем откройте в браузере: `http://localhost:8000`

### Вариант 2: Live Server (VS Code)

1. Установите расширение "Live Server" в VS Code
2. Правой кнопкой на `index.html` → "Open with Live Server"

### Вариант 3: Node.js (http-server)

```bash
npx http-server -p 8000
```

## Настройка

Для изменения контактной информации отредактируйте секцию `#contact` в `index.html`.

Для изменения цветовой схемы отредактируйте переменные цветов в `static/css/style.css`:
- Основной цвет: `#667eea`
- Вторичный цвет: `#764ba2`
