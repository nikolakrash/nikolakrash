  МОДУЛЬ: МАССОВОЕ СОЗДАНИЕ КАНАЛОВ
═══════════════════════════════════════════════════════════════

📋 НАЗНАЧЕНИЕ
───────────────────────────────────────────────────────────────
Модуль позволяет автоматически создавать несколько Telegram-каналов
от выбранных аккаунтов по заранее заданным шаблонам (название, фото,
первое сообщение, закреп и т.п.).

Подходит для запуска сеток каналов под разные тематики.

═══════════════════════════════════════════════════════════════
  ОСНОВНЫЕ НАСТРОЙКИ
═══════════════════════════════════════════════════════════════

1. Количество каналов на аккаунт
   • Сколько каналов будет создано КАЖДЫМ выбранным аккаунтом.
   • Допустимый диапазон: от 1 до 10.

2. Интервал между созданием каналов (от / до, сек)
   • Пауза между запуском создания каналов для разных аккаунтов.
   • Рекомендуется использовать разброс (например 60–180 сек),
     чтобы уменьшить риск FloodWait.

3. Тип канала
   • Публичный — с username (@mychannel), доступен по публичной ссылке.
   • Приватный — без username, доступ по invite-link (t.me/+hash).

4. Очистка системных сообщений
   • Опция удаления системных сообщений (смена фото, username и т.п.)
     после создания канала и установки оформления.

5. Шаблоны каналов
   • Для каждого шаблона задаются:
     - Название канала.
     - Фото канала (опционально).
     - Первое сообщение (с поддержкой Markdown).
     - Медиа к первому сообщению (опционально).
     - Флаг "Закрепить сообщение".
   • Можно добавить несколько шаблонов.
   • При создании канала шаблоны используются по кругу (1, 2, 1, 2, ...).

═══════════════════════════════════════════════════════════════
  АККАУНТЫ И ЛОГИ
═══════════════════════════════════════════════════════════════

• В правом блоке отображается список аккаунтов, доступных для модуля.
  Отмечайте галочками те, с которых нужно создавать каналы.

• Ниже расположен блок "Логи создания каналов":
  - В реальном времени дублирует основные логи процесса:
    запуск создания, паузы между аккаунтами, ошибки, установка фото,
    отправка/закреп первого сообщения, результаты по каждому аккаунту.

═══════════════════════════════════════════════════════════════
  КАК РАБОТАТЬ
═══════════════════════════════════════════════════════════════

1. Выберите аккаунты для работы.
2. Настройте:
   • Количество каналов на аккаунт.
   • Интервал между созданием каналов.
   • Тип канала и опцию удаления системных сообщений.
3. Создайте один или несколько шаблонов (минимум один обязателен).
4. Нажмите "Создать".
5. Следите за ходом выполнения по:
   • Статусу и результатам под формой.
   • Логам в правом блоке.
   • TXT-отчету, который можно скачать после завершения.

═══════════════════════════════════════════════════════════════
  РЕКОМЕНДАЦИИ
═══════════════════════════════════════════════════════════════

• Не создавайте слишком много каналов подряд с одного аккаунта —
  Telegram может выдавать FloodWait.
• Используйте несколько аккаунтов и большие интервалы, если создаете
  крупные сетки.
• Внимательно проверяйте шаблоны перед запуском, чтобы не плодить
  каналы с ошибочными текстами/оформлением.
