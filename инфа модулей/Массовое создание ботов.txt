  МОДУЛЬ: МАССОВОЕ СОЗДАНИЕ БОТОВ
═══════════════════════════════════════════════════════════════

📋 НАЗНАЧЕНИЕ
───────────────────────────────────────────────────────────────
Модуль автоматически создает Telegram-ботов через BotFather
от выбранных аккаунтов по заранее заданным шаблонам (имя, фото, bio).

Подходит для массового запуска сеток ботов под разные проекты.

Важно: модуль работает через официальный BotFather, соблюдайте лимиты
и политику Telegram.

═══════════════════════════════════════════════════════════════
  ОСНОВНЫЕ НАСТРОЙКИ
═══════════════════════════════════════════════════════════════

1. Количество ботов на аккаунт
   • Сколько ботов будет создано КАЖДЫМ выбранным аккаунтом.
   • Диапазон: от 1 до 20.

2. Интервал между созданием ботов (от / до, сек)
   • Пауза между созданием ботов для ОДНОГО аккаунта.
   • Рекомендуется использовать разброс (например 60–180 сек),
     чтобы уменьшить риск FloodWait и ограничений.

3. Шаблоны ботов
   • Для каждого шаблона задаются:
     - Имя бота (отображаемое имя, НЕ username).
     - Фото бота (опционально).
     - Описание (Bio) — опционально.
   • Можно создать несколько шаблонов.
   • При создании ботов шаблоны используются по кругу:
     Шаблон 1, Шаблон 2, Шаблон 1, Шаблон 2, ...

4. Username бота
   • Генерируется автоматически.
   • ВСЕГДА начинается с буквы (ограничение Telegram).
   • Формат: `буква + [буквы/цифры/_] + "_bot"`, длина не более 32 символов.
   • Если username занят или невалиден — модуль пробует другой.

═══════════════════════════════════════════════════════════════
  АККАУНТЫ И ЛОГИ
═══════════════════════════════════════════════════════════════

• В правом блоке «Выбор аккаунтов»:
  - Отображаются аккаунты, доступные для модуля.
  - Можно фильтровать:
    • Без активных задач.
    • Без ограничений.
  - Кнопки:
    • «Выбрать всех» — выделяет все видимые аккаунты.
    • «Снять выбор» — убирает выделение со всех.

• Ниже расположен блок «Логи создания ботов»:
  - В реальном времени показываются:
    • Запуск/завершение для каждого аккаунта.
    • Паузы между аккаунтами (`⏳ Пауза перед запуском аккаунта ...`).
    • Создание каждого бота (имя, попытки username).
    • Все ошибки и предупреждения.

═══════════════════════════════════════════════════════════════
  КАК РАБОТАТЬ
═══════════════════════════════════════════════════════════════

1. Выберите аккаунты, с которых будут создаваться боты.
2. Задайте:
   • Количество ботов на один аккаунт.
   • Интервал между созданием ботов (от / до, сек).
3. Создайте один или несколько шаблонов ботов:
   • Имя (обязательно).
   • Фото (опционально).
   • Bio (опционально).
4. Нажмите «Создать».
5. Наблюдайте за процессом:
   • Статус и результат — в блоке «Результат».
   • Детальные логи — в блоке «Логи создания ботов».
   • После завершения можно скачать TXT-отчет.

═══════════════════════════════════════════════════════════════
  ОТЧЕТ
═══════════════════════════════════════════════════════════════

TXT-отчет содержит:
  • Количество шаблонов.
  • Количество аккаунтов.
  • Количество ботов на аккаунт.
  • Использованный интервал задержек.
  • Список всех созданных ботов в формате:
    `Аккаунт создатель - username бота - токен (ID:TOKEN)`
  • Список ошибок (если были).

Отчет можно скачать по кнопке «Скачать TXT» после завершения работы.

═══════════════════════════════════════════════════════════════
  РЕКОМЕНДАЦИИ
═══════════════════════════════════════════════════════════════

• Не создавайте слишком много ботов подряд с одного аккаунта —
  Telegram может ограничить действия или выдать FloodWait.
• Используйте несколько аккаунтов и большие интервалы.
• Следите за логами: они показывают все ошибки BotFather и проблемы
  с username/лимитами.
• Проверяйте, что аккаунты не в спамблоке и без жестких ограничений.
