ИНВАЙТ ЧЕРЕЗ АДМИНИСТРАТОРОВ — КРАТКАЯ ИНСТРУКЦИЯ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ

Зачем нужен
- Приглашает пользователей в группу через временную выдачу админки.
- Работает циклами и распределяет инвайты по выбранным аккаунтам.
- Показывает прогресс, логи и формирует отчет.

Что делает каждый шаг (1–12)
1) Владелец / главный администратор
   Аккаунт, который управляет группой и выдает права ботам/инвайтерам.

2) Аккаунты-инвайтеры
   Те, кто непосредственно приглашает пользователей. Каждый аккаунт используется
   один раз в цикле. Можно фильтровать и выбирать массово.

3) Целевая группа
   Куда приглашать пользователей. Формат: @group или https://t.me/group.

4) Bot 1
   Бот для выдачи/снятия админки. Нужны username и API токен.

5) Bot 2
   Второй бот для тех же операций (рекомендуется иметь два).

6) Количество активных администраторов
   Сколько аккаунтов-инвайтеров одновременно работают с админкой.

7) Инвайтов с одного аккаунта (от/до)
   Сколько пользователей приглашает каждый аккаунт в одном цикле.

8) Задержка между инвайтами (от/до)
   Интервал между приглашениями, чтобы снизить риск FloodWait.

9) Задержка между стартами аккаунтов (от/до)
   Рандомная задержка, чтобы разнести старт инвайтеров по времени.

10) Максимальный FloodWait (сек)
   Если FloodWait меньше заданного — аккаунт ждет и продолжает.
   Если больше — аккаунт исключается из текущего цикла.

11) Количество циклов
   Сколько раз прогонять список аккаунтов-инвайтеров.

12) Список пользователей для инвайта
   Список username по одному в строке или через запятую.

Функционал и возможности
- Автоматическая выдача/снятие админки через ботов.
- Параллельная работа нескольких инвайтеров.
- Гибкие задержки и лимиты инвайтов.
- Логи в реальном времени.
- Итоговый отчет (TXT) с результатами.

Как работать (коротко)
1) Заполните шаги 1–12.
2) Выберите аккаунты-инвайтеры.
3) Нажмите «Начать инвайт».
4) Следите за логами и прогрессом.
5) После завершения скачайте отчет.

Рекомендации
- Используйте 2–3 аккаунта минимум, чтобы распределить нагрузку.
- Ставьте задержку 10–25 сек и небольшие лимиты инвайтов на аккаунт.
- Для приватных групп заранее добавьте ботов и права вручную, если нужно.
- При большом FloodWait снижайте скорость и количество активных админов.
