АВТОРЕГИСТРАТОР / РУЧНАЯ РЕГИСТРАЦИЯ — КРАТКАЯ ИНСТРУКЦИЯ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ

Зачем нужен
- Модуль помогает быстро регистрировать новые Telegram‑аккаунты:
  - в режиме **авторегистрации** через SMS‑сервисы;
  - в режиме **ручной регистрации**, когда вы сами вводите номер и коды.
- Удобен как для разовых регистраций, так и для массового запуска пачки аккаунтов.

Режимы работы
- **Ручной ввод номера**:
  - Включите переключатель «Ручной ввод номера телефона».
  - Номер и SMS‑код вы вводите сами (например, номер куплен где‑то ещё).
  - Страна/оператор и SMS‑сервис в этом режиме не обязательны.
- **Авторегистрация через SMS‑сервисы**:
  - Переключатель ручного режима выключен.
  - Модуль сам покупает номера и получает коды от выбранного сервиса:
    - 5sim.net, hero-sms.com, grizzlysms.com, vak-sms.com, 365sms.com.

Основные шаги настройки (авторегистрация)
1) **Выбор SMS‑сервиса**  
   - В поле «SMS сервис» выберите площадку, где у вас есть баланс.
2) **API‑ключ выбранного сервиса**  
   - Введите API‑ключ в соответствующее поле (5sim / hero‑sms / grizzlysms / vak‑sms / 365sms).  
   - В подсказке под полем указано, где его взять в личном кабинете.
3) **Опционально: antisafety.net**  
   - Введите `API KEY` и `App ID (AID)`, если хотите:
     - использовать Gmail‑адреса от antisafety;
     - включить push‑уведомления;
     - включить автоматический обход reCaptcha.
   - Без этих настроек модуль всё равно работает, просто с меньшим «качеством» профилей.
4) **Дополнительные настройки (если есть в интерфейсе)**  
   - Гео / страна / оператор для номера.
   - Количество аккаунтов, которые планируете создать за один запуск.

Как запустить
1) Выберите режим:
   - либо включите ручной ввод номера,
   - либо оставьте автозакупку номеров через SMS‑сервис.
2) Заполните все обязательные поля (API‑ключи, гео, доп. параметры).
3) Нажмите кнопку запуска модуля.  
   - В процессе смотрите логи (какой номер куплен, какой код получен, какие ошибки).
4) После завершения:
   - Проверьте, сколько аккаунтов удалось зарегистрировать;
   - При необходимости запустите модуль ещё раз с новыми параметрами.

Рекомендации и ограничения
- Не пытайтесь регистрировать слишком много аккаунтов подряд с одного IP/устройства — высок риск блокировок.
- Следите за балансом и лимитами на выбранных SMS‑сервисах (5sim, hero‑sms и др.).
- Если видите частые ошибки (FloodWait, слишком много попыток, неверные коды) — увеличьте задержки и уменьшите интенсивность.
- Для гео с жёсткими проверками (reCaptcha, обязательная почта) рекомендуется использовать интеграцию с antisafety.net.
