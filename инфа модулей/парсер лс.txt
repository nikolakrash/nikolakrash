МОДУЛЬ «ПАРСЕР ЛИЧНЫХ СООБЩЕНИЙ»
==================================

Этот модуль собирает пользователей из личных сообщений (диалогов) выбранных аккаунтов Telegram. 
Парсер проходит по всем диалогам аккаунта и извлекает контакты, с которыми есть история переписки.

ПРИНЦИП РАБОТЫ
--------------
1. Вы выбираете один или несколько аккаунтов для парсинга.
2. Софт подключается к каждому аккаунту и получает список всех диалогов (личных сообщений).
3. Из каждого диалога извлекается информация о пользователе:
   - User ID (обязательно)
   - Username (если есть)
4. Автоматически пропускаются:
   - Боты
   - Сам аккаунт (себя)
   - Группы и каналы (только личные диалоги с пользователями)
5. Результаты группируются и показываются в интерфейсе.
6. После завершения можно скачать результаты в формате TXT.

НАСТРОЙКИ
---------

1. ВЫБОР АККАУНТОВ ДЛЯ ПАРСИНГА
--------------------------------
- **Список аккаунтов**
  Выберите один или несколько авторизованных аккаунтов, которые будут парсить свои личные сообщения.
  Каждый аккаунт парсит только свои диалоги (тех, с кем он переписывался).

- **Кнопки «Выбрать всех / Снять выбор»**
  Быстро выделить или снять выделение со всех доступных аккаунтов.

Рекомендация:
- Для одного аккаунта парсинг обычно занимает несколько секунд или минут (зависит от количества диалогов).
- При выборе нескольких аккаунтов они обрабатываются последовательно с задержкой между ними.

2. ЗАДЕРЖКА МЕЖДУ АККАУНТАМИ
-----------------------------
- **«От (сек)» / «До (сек)»**
  Случайная задержка между обработкой разных аккаунтов (в секундах).
  После завершения парсинга одного аккаунта делается пауза, затем начинается парсинг следующего.
  
  По умолчанию: 2-5 секунд.
  
Рекомендация:
- Для 1-2 аккаунтов можно оставить минимальные задержки (2-5 сек).
- При большом количестве аккаунтов (5+) увеличьте задержку до 5-10 секунд для безопасности.

РЕЗУЛЬТАТЫ ПАРСИНГА
-------------------
После завершения парсинга отображаются статистика:

- **Всего найдено** — общее количество уникальных пользователей из всех диалогов.
- **С username** — сколько пользователей имеют @username.
- **Только ID** — сколько пользователей без username (только ID).

Примечание:
- Если один и тот же пользователь есть в диалогах нескольких аккаунтов, он считается один раз (уникальные пользователи).
- Результаты можно скачать в формате TXT для дальнейшего использования.

КАК РАБОТАЕТ ПАРСИНГ (ПОШАГОВО)
-------------------------------
1. Вы выбираете аккаунты и настраиваете задержку.
2. Нажимаете «Начать парсинг».
3. Софт последовательно подключается к каждому выбранному аккаунту:
   - Получает список всех диалогов аккаунта.
   - Проходит по каждому диалогу.
   - Извлекает информацию о пользователе (ID, username).
   - Пропускает ботов, группы, каналы и самого себя.
4. После обработки одного аккаунта делается пауза (задержка).
5. Затем обрабатывается следующий аккаунт.
6. Все найденные пользователи собираются в общий список (без дублей).
7. По завершении показываются результаты и кнопка «Скачать результаты (TXT)».

ПРОГРЕСС ПАРСИНГА
-----------------
Во время работы отображается:
- Прогресс-бар с процентом выполнения (обработано аккаунтов / всего аккаунтов).
- Статистика в реальном времени:
  - Всего найдено
  - С username
  - Только ID
- Логи процесса с информацией о каждом этапе.

После завершения блок прогресса автоматически скрывается, остаются только финальные результаты.

РЕКОМЕНДАЦИИ ПО БЕЗОПАСНОСТИ
----------------------------
- Не запускайте парсинг слишком часто с одних и тех же аккаунтов.
- Используйте разумные задержки между аккаунтами (особенно при большом количестве).
- Парсинг личных сообщений — относительно безопасная операция, но не злоупотребляйте.
- Если у аккаунта очень много диалогов (тысячи), парсинг может занять больше времени.

ОТЛИЧИЯ ОТ ПАРСЕРА УЧАСТНИКОВ
------------------------------
- Парсер участников собирает пользователей из групп/каналов.
- Парсер ЛС собирает пользователей из личных диалогов аккаунта.
- В парсере ЛС нет фильтров (активность, телефон и т.п.) — собираются все контакты из диалогов.
- Парсер ЛС работает быстрее, так как не нужно делать запросы к группам.

ПРИМЕНЕНИЕ
----------
Парсер ЛС полезен для:
- Сбора контактов из личных переписок для рассылок.
- Создания базы контактов из диалогов аккаунта.
- Анализа списка людей, с которыми велась переписка.
- Экспорта контактов для дальнейшей работы в других модулях.
