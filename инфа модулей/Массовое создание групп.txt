  МОДУЛЬ: МАССОВОЕ СОЗДАНИЕ ГРУПП
═══════════════════════════════════════════════════════════════

📋 НАЗНАЧЕНИЕ
───────────────────────────────────────────────────────────────
Модуль позволяет автоматически создавать несколько Telegram-групп
от выбранных аккаунтов, по заранее заданным шаблонам (название, фото,
первое сообщение, закреп и т.п.).

Используется для массового запуска сеток групп под разные темы.

═══════════════════════════════════════════════════════════════
  ОСНОВНЫЕ НАСТРОЙКИ
═══════════════════════════════════════════════════════════════

1. Количество групп на аккаунт
   • Сколько групп будет создано КАЖДЫМ выбранным аккаунтом.
   • Допустимый диапазон: от 1 до 10.

2. Интервал между созданием групп (от / до, сек)
   • Пауза между запусками создания групп для разных аккаунтов.
   • Рекомендуется использовать разброс (например 60–180 сек),
     чтобы уменьшить риск FloodWait.

3. Тип группы
   • Публичная — с username (@mygroup), доступна по публичной ссылке.
   • Приватная — без username, доступ по invite-link (t.me/+hash).

4. Дополнительные настройки
   • Анонимные сообщения администратора.
   • Только для чтения (писать могут только админы).
   • Удаление системных сообщений (смена фото, username и т.п.).

5. Шаблоны групп
   • Для каждого шаблона задаются:
     - Название группы.
     - Фото группы (опционально).
     - Первое сообщение (с поддержкой Markdown).
     - Медиа к первому сообщению (опционально).
     - Флаг "Закрепить сообщение".
   • Можно добавить несколько шаблонов.
   • При создании группы шаблоны используются по кругу (1, 2, 1, 2, ...).

═══════════════════════════════════════════════════════════════
  АККАУНТЫ И ЛОГИ
═══════════════════════════════════════════════════════════════

• В правом блоке отображается список аккаунтов, доступных для модуля.
  Отмечайте галочками те, с которых нужно создавать группы.

• Ниже расположен блок "Логи создания групп":
  - Здесь в реальном времени дублируются основные логи процесса:
    создание групп, установка фото, отправка/закреп первого сообщения,
    паузы между аккаунтами, ошибки и т.д.

═══════════════════════════════════════════════════════════════
  КАК РАБОТАТЬ
═══════════════════════════════════════════════════════════════

1. Выберите аккаунты для работы.
2. Настройте:
   • Количество групп на аккаунт.
   • Интервал между созданием групп.
   • Тип группы и дополнительные опции.
3. Создайте один или несколько шаблонов (минимум один обязателен).
4. Нажмите "Создать".
5. Следите за ходом выполнения по:
   • Статусу и результатам под формой.
   • Логам в правом блоке.
   • TXT-отчету, который можно скачать после завершения.

═══════════════════════════════════════════════════════════════
  РЕКОМЕНДАЦИИ
═══════════════════════════════════════════════════════════════

• Не создавайте слишком много групп подряд с одного аккаунта —
  Telegram может выдавать FloodWait.
• Используйте несколько аккаунтов и большие интервалы, если создаете
  крупные сетки.
• Внимательно проверяйте шаблоны перед запуском, чтобы не плодить
  группы с ошибочными текстами/оформлением.
